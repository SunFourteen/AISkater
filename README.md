注意到当前模式的根本缺陷
1. 相似的单词占单词空间的比例极小，正例学习失效
2. 绝大多数单词间拼写相似度为0，模型在绝大多数情况下在推远两个向量，但缺乏更精细的指导
3. 这导致无法使模型学习到有意义的嵌入向量

解决方案
1. 放弃理想化的“全单词”相似
2. 在更细致的层面探讨单词的关系
3. 接下来的重点可能放在找出一种合理的嵌入模式，以“拼写相似”为指导，但不一定会严格按照“拼写相似”进行
